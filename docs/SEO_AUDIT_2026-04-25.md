# SEO/GEO/AEO технический аудит — 2026-04-25

## Что проверено
- Статические SEO-мета и canonical в `index.html`.
- Динамическая SEO-логика React (`src/app/components/SEO.tsx`) для всех роутов.
- Индексация блога для ботов через edge-функции (`functions/blog/[slug].ts`).
- Генерация sitemap/feed и формат дат (`functions/sitemap.xml.ts`, `functions/feed.xml.ts`, `functions/_lib/seo.ts`).
- Robots/LLMs directives (`public/robots.txt`, `public/llms.txt`).
- Источник и fallback-цепочка статей (`functions/_lib/articles.ts`, `scripts/fetch-articles.js`, `src/app/components/hooks/useArticlesApi.ts`).
- Роутинг и доступность URL (`src/app/routes.tsx`, `public/_redirects`, `functions/_middleware.ts`).

## Выявленные риски до фикса
1. **Нестабильный формат дат для sitemap и RSS**
   - Если в данных статьи дата приходит как `DD.MM.YYYY`, sitemap мог отдавать невалидный `<lastmod>`.
   - RSS мог публиковать текущую дату вместо фактической, если дата статьи не парсилась напрямую.
2. **JSON-LD даты могли быть некорректны для некоторых форматов**
   - В `BlogPosting` дата бралась как есть, без нормализации к ISO.

## Что исправлено
- Добавлена унификация дат `toIsoDate()` с поддержкой ISO и `DD.MM.YYYY`.
- В sitemap `<lastmod>` теперь всегда валидный `YYYY-MM-DD` (или fallback на текущую дату).
- В RSS `pubDate` строится из нормализованной даты статьи.
- В JSON-LD `datePublished/dateModified` публикуются только в валидном формате.

## Текущее состояние индексации (по коду)
- `/blog` и `/blog/:slug` доступны как статически сгенерированные HTML на этапе build (`scripts/generate-pages.js`).
- Для ботов на `/blog/:slug` есть отдельный SEO HTML на edge (`functions/blog/[slug].ts`) с:
  - `<title>`, `<meta description>`, OG/Twitter;
  - canonical;
  - JSON-LD: `BlogPosting`, `BreadcrumbList`, опционально `FAQPage`.
- Sitemap содержит статические роуты + статьи (`functions/sitemap.xml.ts`).
- Robots разрешает индексирование, кроме `/admin` (`public/robots.txt`).
- RSS feed присутствует (`functions/feed.xml.ts`) и подходит для ускоренного discovery.
